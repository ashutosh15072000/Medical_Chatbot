# Medical ChatBot

## Step to run the Project

### Create a new conda environment named 'mchatbot' with Python 3.9
```bash
conda create -p  mchatbot python=3.9 -y
```
### Activate the 'mchatbot' environment
```bash
conda activate mchatbot
```
